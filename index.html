The Act itself:

You can find the official information about the online safety act HERE
In summary, the act aims to protect children from 'adult' content online. It requires users of sites providing this content to send photos of their ID to a third party to verify their age.

There are a number of reasons as to why the act is a bad idea, which I will detail here later, here's a summary:

  • The wording is vague, and captures far more than just porn. This reddit thread summarises some affected subreddits
  • Some subreddits include ones which provider support for alcoholics, victims of sexual assault amoung others.
  • It even took down a forum about hamsters of all things.
  • Wikipedia might be affected by the act
  • General Freedom of Speech + Freedom of the Internet issues
  • Easily Bypassed as detailed below (which may lead to other more restrictive laws like banning VPNs)
  • Massive security risks with 3rd party companies handling our IDs. See TEA App as an example, in which users IDs and GPS Coordinates got leaked.

VPNs

One way to bypass the OSA is to use a VPN. A VPN is a tool which makes it appear as if your internet traffic is coming from another location. Good VPNs will let you choose this location by country, so you can appear to be outside of the UK.
This will let you access blocked content without having to provide you ID to some random company and risk the security breach that comes with that.
IMPORTANT: While there are many VPN services out there, they are not all created equal. Most importantly, do NOT use a free VPN. This is a massive security risk in itself.

Only browser traffic:

If you only need to send your browser traffic through the VPN, then you can use the VPN providers browser extension. This will depend on whether your VPN provider has a browser extension for your particular browser.
The 2 exmaple providers above both provider extensions for the firefox browser.
Once installed, you can enable the VPN via the extension, just make sure to select another country. Try to select one that relatively close for the fastest speeds. France for example.

All device traffic:

Sometimes you want to send all traffic from your machine over the VPN. This might be because you are using a mobile for example, which generally doesn't support browser extensions.
It might be that you need an app like discord to go over the VPN. Discord requires age verification if you are in any server they deem to be NSFW, regardless of if you engage in that content.
In these instances, your VPN provider hopefully supports some desktop or mobile app you can download. Just install that, launch it, and set the country.

Self-hosting VPN endpoint:

I'm not going to detail all the setup required for this here, but wanted to make a note of it for one key reason: The government will know that VPNs will bypass OSA, so may try to enact some law or amend the current act to block VPNs as well.
If this is the case, the best thing you can do is setup a personal private VPN for your own use. Since this is private, it becomes much harder to detect and block.
My approach to this is to setup some sort of VPS - make sure this is hosted outside of the UK. Oracle Cloud lets you run 2 low-tier vms for free which are more than enough. Once you have this setup, run some VPN application like WireGuard or OpenVPN. Both WireGuard and OpenVPN have mobile apps, as well as desktop apps.
